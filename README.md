# Weaviate-Practice

Practice using Weaviates Python client to manage a WCS instance of Weaviate

Goals:
1. Build a Weaviate vector database
2. Query it with:
    - Semantic search
    - Added filters
    - generative searches to transform search results with a large language model (LLM).

Using:<br/>
